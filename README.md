# MAX Autopost (Free) v1.7.8

Fix релиз: MAX upload step2 иногда возвращает payload не как `{token,url,type}`, а как вложенный объект, например `{"photos":{...}}`.  
Плагин теперь принимает **любой валидный JSON** и передаёт его целиком в `image.payload`.

## Что нового в 1.7.8
- Возвращена корректная работа с кастомными типами записей: хуки row/bulk actions теперь регистрируются после инициализации CPT (`init`), а не слишком рано.
- Убрано преждевременное кэширование списков типов записей, из-за которого часть CPT могла не попадать в список.

## Что нового в 1.7.7
- Внутренний рефакторинг и оптимизация: вынесена нормализация лимита текста и режима источника изображения в отдельные функции, чтобы исключить дублирование логики.
- Оптимизирована обработка очереди: добавлены безопасное снятие lock через `finally` и лёгкие параметры `WP_Query` (`no_found_rows`, отключение лишних кэшей).
- Небольшая оптимизация admin/runtime: добавлено кэширование доступных/поддерживаемых типов записей в рамках запроса.

## Что нового в 1.7.6
- Добавлен выбор источника изображения: из новости, только из новости (без подмены), либо всегда изображение сайта.
- В режиме «только изображение новости» плагин не подставляет изображение сайта, если у записи нет своей картинки.

## Что нового в 1.7.5
- Из вкладки «Техпомощь» окончательно убраны любые визуальные блоки со скриншотами: оставлена только текстовая инструкция и таблица найденных Chat ID.

## Что нового в 1.7.4
- Улучшено обрезание длинного текста: теперь при ограничении по символам добавляется аккуратное многоточие вместо «жёсткого» обрыва.
- Для подсчёта длины используется явная UTF-8 обработка, чтобы избежать некорректного усечения мультиязычного текста.

## Что нового в 1.7.3
- Уменьшен размер автоматической отправки очереди: теперь за один cron-проход отправляется 1 запись (вместо 5), чтобы избежать "залпов" сообщений.
- Улучшено влияние настройки длины текста: размер анонса теперь рассчитывается динамически от выбранного лимита.

## Что нового в 1.7.2

## Features
- One message: image (optional) + text + button (optional)
- Attachments order: image first, button second
- WP-Cron queue + retry/backoff + cron lock
- Logs tab

## Install
Upload folder `max-autopost` into `/wp-content/plugins/`, activate.
